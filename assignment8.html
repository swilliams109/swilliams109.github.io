SQL Commands Used

  • SELECT FROM Customers;
  • SELECT ProductName FROM Products WHERE CategoryID = 2;
  • SELECT COUNT(*) FROM Orders WHERE ShippedDate IS NOT NULL;
  • INSERT INTO Shippers (ShipperID, ShipperName, Phone) VALUES (4, 'On Time Delivery', '(503) 555 0123');
  • UPDATE Products SET Price = Price + 1;
  • UPDATE Products SET Price = Price - 1;
  • UPDATE Shippers SET ShipperName = 'On-Time Delivery' WHERE ShipperID = 4;
  • DELETE FROM Shippers WHERE ShipperID = 4;

Lesson Summary

This lesson helped me understand how SQL commands interact with a database. I practiced using SELECT, WHERE, COUNT, INSERT INTO, UPDATE, and DELETE statements to retrieve, modify, and remove data. These skills are essential for managing a real-world database like the one I plan to use for my cupcake business. I learned how data can be filtered and updated efficiently, which helps in making decisions based on product pricing, order tracking, and customer data. The hands-on activities made it easier to grasp the power of SQL in day-to-day business operations.
